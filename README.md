Installing npm and Node.js
====================
```bash

curl -sL https://deb.nodesource.com/setup | sudo bash -
sudo apt-get install -y nodejs

```

Update npm
==

```bash
sudo npm install npm -g
```


Install dependencies
==
```bash
npm install
```

Run
==
```bash
npm start
```
Run the server (for mosaic):

    $ node server.js
